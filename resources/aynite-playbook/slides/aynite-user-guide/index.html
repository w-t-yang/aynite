Working with Files

File Browser

Your main workspace for viewing, editing, and managing files — split into a file tree and content area.

  • Click any file to open in a tab
  • Ctrl+Tab — Fuzzy-find files instantly
  • Multiple tabs with drag-to-reorder

Three Viewing Modes

🎨 FileView Mode

Specialized renderers for Markdown, PDF, images, audio, video, HTML

👁️ View Mode

Read-only text with syntax highlighting

✏️ Edit Mode

Writable editor with syntax highlighting

File Types & Viewers

📝 Markdown

Rendered with headings, code blocks, images, and local link resolution

📄 PDF

Full PDF rendering with zoom, fit-to-height, and scroll

🖼️ Images

PNG, JPG, GIF, SVG with zoom controls and file info

🎵 Audio

MP3, WAV, FLAC with player controls and streaming

🎬 Video

MP4, WebM, MOV with zoom and streaming support

🌐 HTML

Rendered webpage in an iframe with 3-mode toggle

Switch modes using the bottom toolbar — FileView → View → Edit

Editing & Search

✏️ Editing

  • Ctrl+S — Save file
  • 🟡 Dirty indicator on tab
  • Syntax highlighting by language
  • Only for text-based files

🔍 Search

  • Ctrl+F — Open search bar
  • Ctrl+N — Next match
  • Ctrl+P — Previous match
  • Highlights & match counter

Search highlights are embedded into syntax highlighting — matches stay colored correctly

DataViews — Schemas & Usage

ViewRendersJSON Schema
📊 ChartBar, line, area, pie, radar{keys: string[], data: object[]}
📈 StockCandlestick charts{symbol, data: [{open,high,low,close,volume}]}
🕸️ GraphNetwork graphs{nodes: [{id,label}], links: [{source,target}]}
🔀 FlowNode-based flow editor{nodes: [{position,data}], edges: [{source,target}]}
📐 DiagramMermaid diagrams{definition: string}
🌳 MindMapTree mind maps{root: {id,label,children?}}
✏️ CanvasExcalidraw whiteboard{elements: object[], appState: object}
🎨 ThemeColor theme editor{id: string, colors: {string: string}}

Open any .json file — schema auto-detection switches to the matching dataview renderer

Create Your Own

/create-theme make me a cozy autumn sunset theme

The AI generates a complete theme with all 32 CSS color properties — backgrounds, text, borders, accents, scrollbars, fonts

Saved to ~/.aynite/themes/<name>.json

Press Ctrl+R to refresh the tile

Keybindings & Layout

⌨️ Keybindings

  • Ctrl+R Refresh tile
  • Ctrl+Tab Fuzzy-find files
  • Ctrl+B Toggle left panel
  • Ctrl+I Toggle right panel
  • Ctrl+L Focus chat
  • Ctrl+S Save file (edit mode)
  • Ctrl+F Search in file

🧩 Tile Layout

  • Ctrl+- Split vertically
  • Ctrl+= Split horizontally
  • Ctrl+Q Close tile
  • Ctrl+O Cycle tiles
  • Ctrl+Shift+←→↑↓ Resize tiles

Customize in Settings → Keybindings

Pro tip: Load any view in an empty tile — or start with pre-built layouts
