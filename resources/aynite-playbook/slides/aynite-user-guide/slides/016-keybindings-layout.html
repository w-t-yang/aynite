Keybindings & Layout

⌨️ Keybindings

  • Ctrl+R Refresh tile
  • Ctrl+Tab Fuzzy-find files
  • Ctrl+B Toggle left panel
  • Ctrl+I Toggle right panel
  • Ctrl+L Focus chat
  • Ctrl+S Save file (edit mode)
  • Ctrl+F Search in file

🧩 Tile Layout

  • Ctrl+- Split vertically
  • Ctrl+= Split horizontally
  • Ctrl+Q Close tile
  • Ctrl+O Cycle tiles
  • Ctrl+Shift+←→↑↓ Resize tiles

Customize in Settings → Keybindings

Pro tip: Load any view in an empty tile — or start with pre-built layouts
